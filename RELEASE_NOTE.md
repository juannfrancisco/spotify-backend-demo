# Versión 1.0.0

## Descripción

Versión inicial estable del repositorio **znt-template-angular-tailwind-front**.

### Características principales:
- Proyecto Angular configurado con TailwindCSS para estilos modernos y personalizables.
- Dockerfile multistage: build de Angular y despliegue en NGINX.
- Variables de entorno para integración con backend y autenticación Google.
- Instalación eficiente de dependencias con `npm ci`.
- Configuración personalizada de NGINX para servir la aplicación.
- Listo para despliegue en contenedores y plataformas cloud.

Esta versión sirve como base para nuevos proyectos frontend en ZENTA, facilitando el desarrollo, despliegue y mantenimiento.
